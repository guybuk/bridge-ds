Welcome to the bridge-ds documentation!
=======================================

*Bridge* is a lightweight Python framework designed to provide a
unified interface to deep learning datasets from different
modalities.

Perform global operations, aggregations and queries
with a Pandas-like experience, and handle individual samples and raw data using a
class-based, easy-to-work-with interface.
